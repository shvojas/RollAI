# RollAI
RollAI is an AI-powered inspection system using YOLOv8 for detecting and classifying six types of surface defects in rolled steel. Trained on the NEU dataset, it achieves 88% mAP with real-time visualization via Streamlit, enabling fast, accurate, and automated industrial quality control.
